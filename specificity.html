Specificity

Specificity adalah setiap deklarasi css (selector) memiliki berat yang berbeda. Berat tersebut menentukan seberapa spesifik sebuah elemen dapat dipilih oleh selector. Contoh perhitungannya:

Contoh Inline Id Class Elemen
ul#sarapan li 0 1 0 2
ul#sarapan li.favorit 0 1 1 2
