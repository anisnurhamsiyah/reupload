Font Styling

Font style adalah style yang terkait dengan penulisannya. Macam-macam property yang digunakan font dalam CSS:

  • Font-family, digunakan untuk mengatur jenis font yang akan digunakan (nama font, generic family).
  • Font-size, digunakan untuk mengatur ukuran font (px, %, em).
  • Font-weight, digunakan untuk mengatur ketebalan font (normal, bold).
  • Font-variant, digunakan untuk mengubah font menjadi small caps (normal, small caps).
  • Font-style, digunakan untuk mengubah font menjadi bercetak miring (normal, italic, oblique).
  • Line-height, digunakan untuk mengatur spasi antar baris (px, em).
